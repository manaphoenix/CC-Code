Overview

The Linq Library brings the power of .NET's LINQ (Language Integrated Query) to Lua. It provides a comprehensive set of methods for querying and manipulating tables with a fluent, chainable interface. The library also supports lambda expressions for concise and expressive query logic.

Installation

wget https://raw.githubusercontent.com/manaphoenix/CC_OC-Code/refs/heads/main/lib/linq.lua

Basic Usage

Create a linq table and start using the collection methods:

lua 
local linq = require("linq")

local numbers = linq({1, 2, 3, 4, 5})
local evens = numbers:where("x => x % 2 == 0")
local doubled = evens:select("x => x * 2")
-- Result: {4, 8}

Methods

concat(other)

Combines two tables into one.

lua 
local allPeople = group1:concat(group2)

where(predicate)

Filters elements based on a predicate.

lua 
local adults = people:where("p => p.age >= 18")

select(selector)

Transforms elements using a selector function.

lua 
local names = people:select("p => p.name")

contains(value)

Checks if a value is present in the collection.

lua 
local hasJohn = people:contains("John")

intersect(other)

Returns the common elements between two collections.

lua 
local common = group1:intersect(group2)

foreach(action)

Executes an action for each element in the collection.

lua 
people:foreach("p => print(p.name)")

removeAll(predicate)

Removes all elements that match the predicate.

lua 
people:removeAll("p => p.age < 18")

remove(value)

Removes a specific value from the collection.

lua 
people:remove("John")

count()

Returns the number of elements in the collection.

lua 
local total = people:count()

all(predicate)

Checks if all elements satisfy the predicate.

lua 
local allAdults = people:all("p => p.age >= 18")

any(predicate)

Checks if any element satisfies the predicate.

lua 
local hasAdults = people:any("p => p.age >= 18")

except(other)

Returns elements that are not in the other collection.

lua 
local unique = group1:except(group2)

last([predicate])

Gets the last element meeting the predicate.

lua 
local lastAdult = people:last("p => p.age >= 18")

max(selector)

Finds the maximum value according to a selector function.

lua 
local oldest = people:max("p => p.age")

min(selector)

Finds the minimum value according to a selector function.

lua 
local youngest = people:min("p => p.age")

orderBy(selector)

Orders elements based on a selector function.

lua 
local sorted = people:orderBy("p => p.age")

reverse()

Reverses the order of elements in the collection.

lua 
local reversed = people:reverse()

skip(count)

Skips a specified number of elements in the collection.

lua 
local remaining = people:skip(2)

Examples

Filtering and Transforming

lua 
local data = linq({
    {name = "John", age = 25},
    {name = "Jane", age = 17},
    {name = "Bob", age = 30}
})

local adultNames = data
    :where("p => p.age >= 18")
    :select("p => p.name")
-- Result: {"John", "Bob"}

Advanced Usage

Complex Queries

lua 
-- Assuming linq.lua is properly required
local linq = require("lib.linq")

-- Sample data: a list of products
local products = {
    { Name = "Blender",    Category = "Appliances",  Price = 29.99 },
    { Name = "Toaster",    Category = "Appliances",  Price = 19.99 },
    { Name = "Laptop",     Category = "Electronics", Price = 999.99 },
    { Name = "Smartphone", Category = "Electronics", Price = 699.99 },
    { Name = "Desk",       Category = "Furniture",   Price = 199.99 },
    { Name = "Chair",      Category = "Furniture",   Price = 89.99 },
}

-- LINQ-like query
local query = linq(products)
    :where(function(product) return product.Price > 50 end) -- Filter products with price greater than 50
    :groupBy(function(product) return product.Category end) -- Group by category
    :select(function(group)
        return {
            Category = group.Key,
            Products = linq(group):orderBy(function(product) return product.Price end):toList(), -- Order products by price
            TotalPrice = linq(group):sum(function(product) return product.Price end)             -- Calculate total price for the group
        }
    end)

-- Print the results
for _, categoryGroup in ipairs(query:toList()) do
    print("Category: " .. categoryGroup.Category)
    print("Total Price: " .. categoryGroup.TotalPrice)
    for _, product in ipairs(categoryGroup.Products) do
        print("  Product: " .. product.Name .. ", Price: " .. product.Price)
    end
end
